# GeoAttendance API

### Get Status
`GET https://api..com`**/**

Returns the status of the GeoAttendance API.

#### Body
    
- **status**  *string*
    - Current status of the server
- **stats** *object*
    - Current inserver stats
        - **users** *integer* 
            - Amount of users signed up

        "stats": {
            "users": 0,
        }
